# Netflix Content Analysis - Exploratory Data Analysis (EDA)

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset. The objective is to analyze Netflix content, identify trends and patterns, visualize key findings, and generate data-driven insights using Python.

This project was completed as part of the AI & Machine Learning Internship Program.

---

## Dataset

**Dataset:** Netflix Movies and TV Shows Dataset

The dataset contains information about Netflix content, including:

* Title
* Type (Movie / TV Show)
* Director
* Cast
* Country
* Date Added
* Release Year
* Rating
* Duration
* Genre

---

## Objectives

* Load and inspect the dataset
* Perform data cleaning and preprocessing
* Analyze content distribution and trends
* Create meaningful visualizations
* Generate business insights from the data

---

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

---

## Data Cleaning Performed

* Handled missing values in director, cast, country, and rating columns
* Removed duplicate records
* Converted date columns to appropriate data types
* Removed irrelevant columns where necessary
* Verified data quality before analysis

---

## Exploratory Data Analysis

The following questions were analyzed:

1. What is the distribution of Movies and TV Shows?
2. Which countries contribute the most content?
3. What are the most common content ratings?
4. How has Netflix content grown over time?
5. Which directors have the highest number of titles?

---

## Visualizations

The project includes the following visualizations:

* Bar Chart
* Pie Chart
* Histogram
* Line Chart
* Scatter Plot
* Heatmap

All visualizations include appropriate titles and labels for readability.

---

## Key Insights

* Movies significantly outnumber TV Shows on Netflix.
* The United States contributes the highest amount of content.
* TV-MA is the most common content rating.
* Netflix content production increased rapidly after 2015.
* A small number of directors contribute a large share of titles.
